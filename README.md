# AUTO-EVO-AI V0.1

## 一句话
> 说一句话，系统自动完成几乎所有工作。

## 快速开始
```bash
# 启动
python api_server.py --port 8765

# 浏览器访问
http://localhost:8765/
```

## 核心能力
- **497个自动化模块**: 语音/OCR/知识图谱/RAG/视频/翻译/代码审查/CI-CD/浏览器自动化/智能家居...
- **8个全自动工作流**: 文档处理/代码部署/数据分析/股票分析/网页操作/会议/家居/翻译
- **2077个n8n工作流**: 可搜索、触发、扩展
- **MCP聚合网关**: 连接Smithery(7300+工具)/mcp.so/Glama
- **万能Skill转换**: 任意GitHub仓库→系统Skill
- **266位AI专家**: 覆盖18个部门

## 架构
```
frontend/         聊天界面(chat.html)
api/              后端API路由(183个路由文件)
modules/          业务模块(497个)
core/             核心引擎
skills/           Skill系统
```

## 系统要求
- Python 3.10+
- 腾讯云/香港服务器（推荐）
