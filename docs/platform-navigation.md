## 一、顶部导航栏

```
┌──────────────────────────────────────────────────────────────┐
│  ⚡ EVO-AI     💬 对话    🪐 开源中心  📊 虚拟公司  🔧 工作台  │
│                                         🔔  👤 admin        │

│                                                              │
│                  当前工作区内容...                             │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

| 导航 | 快捷键 | 工作区 |
|------|--------|--------|
| 💬 对话 | `Alt+1` | 聊天/助手 |
| 🪐 开源中心 | `Alt+2` | 发现/集成/画布 |
| 📊 虚拟公司 | `Alt+3` | 仪表盘/部门/Agent |
| 🗂️ 我的项目 | `Alt+4` | 已集成项目/监控 |
| 🔧 工作台 | `Alt+5` | 开发/编排/设置 |
| 🏪 市场 | `Alt+6` | 模板/社区/分享 |
| ⚙️ 管理 | `Alt+7` | 系统/用户/配置 |

## 二、七大工作区详解

### 1. 💬 对话 — 聊天助手区

**适用场景**：日常使用、快速提问、随意探索

```
布局:
┌──────────────────────────────────────┐
│  💬 对话   📋 历史   🔄 新对话  🎤 语音 │
├──────────────────────────────────────┤
│                                      │
│  ┌──────────────────────────────┐    │
│  │ 用户: 帮我写一封邮件...       │    │
│  └──────────────────────────────┘    │
│  ┌──────────────────────────────┐    │
│  │ AI: 好的，以下是邮件草稿...    │    │
│  └──────────────────────────────┘    │
│                                      │
│  ┌─ 快捷工具 ───────────────────┐    │
│  │ 浏览器/搜索/绘图/翻译/代码   │    │
│  └──────────────────────────────┘    │
│  ┌──────────────────────────────┐    │
│  │ 输入你想做的事...        ➤🎤 │    │
│  └──────────────────────────────┘    │
└──────────────────────────────────────┘
```

**功能**：
- LLM 对话（Qwen3.6/R1 自动路由）
- 80+ 快捷工具按钮
- 语音输入
- 历史记录
- 多语言（9种语言）

### 2. 🪐 开源中心 — 发现与集成区

**适用场景**：寻找新项目、集成、组合

```
子页面:
├─ 🔍 发现 (Discover)     ← 默认页

│   ├─ 分类筛选 (AI/Web/数据/工具...)
│   └─ 搜索框
│
├─ 📦 我的项目 (My Projects)
│   ├─ 已集成项目列表
│   ├─ 状态/资源/端口概览
│   └─ 批量操作
│
├─ 🎨 画布 (Canvas)        ← 核心
│   ├─ 节点拖拽
│   ├─ 连线组合
│   ├─ 属性面板
│   └─ 部署/导出
│
└─ 🧬 衍生 (Forks)
    ├─ 衍生树
    ├─ 版本历史
    └─ 发布管理
```

## 三、工作区流转路径

```
用户首次打开
    ↓
💬 对话 (默认) → 了解系统能力
    ↓
🪐 开源中心 → 发现并集成项目
    ↓
🗂️ 我的项目 → 管理已集成内容
    ↓
🔧 工作台 → 组合/二次开发
    ↓
🏪 市场 → 分享模板
    ↓
📊 虚拟公司 → AI员工协同
    ↓
⚙️ 管理 → 系统配置
```
