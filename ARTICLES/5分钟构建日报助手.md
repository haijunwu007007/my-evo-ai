# 用 AUTO-EVO-AI 5分钟构建一个能自动进化的日报助手

> 上市公司级 AI 自动化系统，从零到 Dashboard 只需 3 条命令。

## 现状：日报是很多团队的刚需

每天花30分钟汇总代码提交、GitHub热门趋势、系统健康数据，做成日报发给团队——这个工作完全应该自动化。

## 解决方案

AUTO-EVO-AI 内置 536 个生产模块，直接调用即可：

```bash
# 1. 克隆
git clone https://github.com/haijunwu007007/my-evo-ai.git
cd my-evo-ai

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动
python api_server.py

# 打开 http://127.0.0.1:8765/dashboard
```

## 日报怎么生成

系统已预置模板任务：

| 任务 | 频率 | 数据源 |
|------|------|--------|
| GitHub Trending 分析 | 每4小时 | Search API |
| 系统健康巡检 | 每小时 | 全模块 |
| 日常自我进化 | 每天9:00 | 趋势+分析+报告 |

访问 `http://127.0.0.1:8765/dashboard` 在"系统洞察"面板查看。

## 完整日报用例

`examples/daily-report-agent/README.md` 包含：
- 从 API 获取 GitHub Python 热门项目 Top5
- 统计系统模块健康状态和调度任务执行情况
- 输出 Markdown 格式日报

## 架构概览

```
API Server (:8765) → 536 懒加载模块 → 4 引擎(调度/事件/管线/队列)
                  → SQLite 持久化 → JWT 认证 → Prometheus 指标
```
