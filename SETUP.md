# 🚀 AUTO-EVO-AI V0.1 — 部署选择指南

## 你需要什么？

> **本系统需要一个后台服务器来运行。** 你的手机/平板/电脑通过浏览器访问它。
> 下面两种方案，选一种最适合你的。

---

## 方案 A：用家里现有的电脑（免费）

**适合：** 家里有台电脑平时开着机，偶尔用一下

### 准备工作

| 需要 | 说明 |
|------|------|
| ✅ Windows 电脑 | 能正常开机、联网 |
| ✅ 手机 | 浏览器即可，iPhone/Android 都行 |
| ⏱ 第一次设置：10 分钟 | 后续开机自动运行 |

### 设置步骤

**第 1 步：下载本系统**
```
将 D:\AUTO-EVO-AI-V0.1 整个文件夹放到电脑上
```

**第 2 步：首次运行**
右键 `start.ps1` → **以管理员身份运行**
（自动开防火墙 + 启动服务）

**第 3 步：在手机上装 Tailscale**

| 手机 | 操作 |
|------|------|
| iPhone | App Store 搜索 "Tailscale" 安装 |
| Android | 应用商店搜索 "Tailscale" 安装 |

电脑和手机都装好 Tailscale 后，登录同一个账号。

**第 4 步：手机访问**

打开手机浏览器，输入电脑的 Tailscale IP + `:8765`：
```
http://100.x.x.x:8765/app/login
```

> ⚠️ 电脑必须开机 + 连着网 + 跑着服务。关机了手机就连不上。

---

## 方案 B：租一台云服务器（约 30 元/月）

**适合：** 天天都用、移动办公、不想管电脑开关机

### 准备工作

| 需要 | 花钱吗 |
|------|--------|
| ✅ 云服务器 | **约 30 元/月**（腾讯云/阿里云最便宜套餐） |
| ✅ 手机 | 浏览器即可 |
| ⏱ 第一次设置：15 分钟 | 我提供一键部署脚本 |

### 设置步骤

**第 1 步：买云服务器**

| 平台 | 最低配置 | 价格 |
|------|---------|------|
| [腾讯云](https://cloud.tencent.com) 轻量应用服务器 | 2核2G | **约 30 元/月** |
| [阿里云](https://www.aliyun.com) ECS | 2核2G | **约 32 元/月** |

操作系统选 **Ubuntu 22.04** 或 **CentOS 7+**。

**第 2 步：一键部署**

买好服务器后，把 IP 发给我们，运行部署脚本即可自动装好本系统。

**第 3 步：手机访问**

打开手机浏览器，输入云服务器 IP：
```
http://<服务器IP>:8765/app/login
```
首次访问自动进入设置向导 → 创建账号 → 配 API Key → 开始使用。

**优点：电脑不用开着，手机随时能用。**

---

## 两种方案对比

| 对比项 | 🏠 方案 A：家里电脑 | ☁️ 方案 B：云服务器 |
|--------|-------------------|-------------------|
| **费用** | 免费（电费忽略） | **约 30 元/月** |
| **手机随时能用** | ⚠️ 电脑必须开机 | ✅ 24小时在线 |
| **设置难度** | ⭐⭐ 中等 | ⭐ 简单（一键部署） |
| **速度** | 取决于家里网速 | 取决于云服务器带宽 |
| **数据安全** | 数据在自己电脑 | 数据在云端 |
| **适合谁** | 偶尔用用、不介意电脑开着 | 天天用、移动办公、省心 |

---

## 下一步

- **选方案 A** → 运行 `start.ps1` + 装 Tailscale
- **选方案 B** → 买云服务器，把 IP 发给我们部署
- **不确定** → 先方案 A 免费试用，觉得好用再升级方案 B

> 💡 遇到任何问题，重启电脑后运行 `start.ps1` 即可恢复。

## 附：本系统的两种界面

| 版本 | 地址 | 说明 |
|------|------|------|
| **🆕 新版 SPA** | `http://IP:8765/app/login` | **推荐。** 完整功能：Dashboard/模块树/外部工具/Agent-S/语音输入/主题切换/设置向导 |
| **🕰️ 旧版 Dashboard** | `http://IP:8765/dashboard` | 早期的单文件 HTML 版本，仅显示模块列表，功能有限 |

**关系：** 旧版是新版的"前身"。新版 SPA 覆盖了旧版所有功能，并增加了登录认证、模块管理、外部工具集成、语音输入等大量新能力。**直接使用新版即可。**

> ⚠️ 不要用浏览器直接双击打开 `index.html`（`file:///D:/.../index.html`），那样 JavaScript 无法连接到 API 服务器，页面是空的。必须通过 `http://localhost:8765/dashboard` 访问。
