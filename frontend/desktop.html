🪟 Windows 安装步骤

1Chrome 用户：右上角菜单 → 更多工具 → 创建快捷方式 → 勾选「打开为窗口」
2Edge 用户：右上角菜单 → 应用 → 将此站点安装为应用
3安装后桌面会出现图标，双击即可打开
📌 推荐使用 Chrome，安装后像原生应用一样运行

🍎 macOS 安装步骤

1Safari 用户：菜单栏 → 文件 → 添加到程序坞
2Chrome 用户：右上角菜单 → 更多工具 → 创建快捷方式 → 勾选「打开为窗口」
3程序坞或桌面出现图标，点击即可打开

📱 iOS (iPhone/iPad) 安装步骤

1使用 Safari 浏览器 打开此页面（其他浏览器不支持）
2点击底部 分享按钮（方框+箭头图标）
3向下滑动，选择 「添加到主屏幕」
4点右上角 「添加」，桌面出现图标

🤖 Android 安装步骤

1使用 Chrome 浏览器 打开此页面
2右上角 三点菜单 → 「添加到主屏幕」
3底部弹出安装确认 → 点 「安装」
4桌面出现图标，点击即可打开

🌐 安装说明

请使用 Chrome 或 Edge 浏览器打开此页面安装。
安装后像原生应用一样运行，支持离线缓存和桌面通知。
