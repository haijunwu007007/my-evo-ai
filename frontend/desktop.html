🪟 Windows 一键安装

  • 1点击下方按钮，下载 install-evo.bat
  • 2双击下载的文件运行
  • 3桌面出现 AUTO-EVO-AI 图标

🪟 Windows 安装

  • 1点击「下载安装脚本」按钮
  • 2双击下载的 install-evo.bat
  • 3桌面出现 AUTO-EVO-AI 图标，双击打开

🍎 macOS 安装

  • 1用 Safari 打开此页面
  • 2菜单栏 → 文件 → 添加到程序坞

📱 iOS 安装

  • 1用 Safari 打开此页面
  • 2点底部分享按钮 → 添加到主屏幕

🤖 Android 安装

  • 1用 Chrome 打开此页面
  • 2右上角菜单 → 添加到主屏幕
