🖥️ 检测到 Windows 推荐

  • 1点击下方按钮，下载安装脚本
  • 2双击下载的 install-evo.bat
  • 3桌面出现 AUTO-EVO-AI 图标，双击打开
⚡ 扫码也可安装 · 手机相机扫描下方二维码即可访问

🪟 Windows 安装步骤

  • 1点击上方「下载安装脚本」按钮
  • 2保存 install-evo.bat 到桌面
  • 3双击运行，自动创建桌面快捷方式
  • 4双击桌面 AUTO-EVO-AI 图标打开
💡 如果用 Edge 浏览器：右上角菜单 → 应用 → 将此站点安装为应用

🍎 macOS 安装步骤

  • 1用 Safari 打开此页面
  • 2菜单栏 → 文件 → 添加到程序坞
  • 3程序坞出现图标，点击即可打开

📱 iOS 安装步骤

  • 1用 Safari 打开此页面
  • 2点击底部 分享按钮（方框+箭头）
  • 3向下滑 → 点 「添加到主屏幕」
  • 4点右上角「添加」→ 桌面出现图标

🤖 Android 安装步骤

  • 1用 Chrome 打开此页面
  • 2右上角 三点菜单 → 「添加到主屏幕」
  • 3底部弹出安装确认 → 点「安装」
  • 4桌面出现图标，点击即打开
