📋 安装步骤

1

准备服务器

需要一台 Linux 服务器（Ubuntu 22.04+），已安装 Docker。

2

一键安装

SSH 登录服务器，执行以下命令：

curl -s https://autoevoai.com/install/install.sh | bash📋 点击复制
3

等待启动

脚本会自动下载 Docker 镜像并启动服务。首次启动约 30-60 秒。

docker logs -f evo-api📋 点击复制
4

打开浏览器

访问 http://你的服务器IP:8765 进入系统。
如需 HTTPS + 域名，参考：

# 使用 Caddy 自动 HTTPS\necho "autoevoai.com {\n reverse_proxy :8765\n}" > Caddyfile && docker run -d -p 80:80 -p 443:443 -v $PWD/Caddyfile:/etc/caddy/Caddyfile caddy📋 点击复制
5

开始使用

首次登录会显示引导弹窗，选 3 个问题即可自动配置推荐场景。

🚀 进入系统
